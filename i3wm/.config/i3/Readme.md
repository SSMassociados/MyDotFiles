## Estrutura Organizada

~/.config/i3/
├── config (main.conf)
└── modules/
    ├── env.conf               # Variáveis e constantes
    ├── programs.conf      # Definições de programas
    ├── decoration.conf     # Aparência e cores
    ├── input.conf              # Mouse e dispositivos
    ├── monitors.conf        # Monitores e outputs
    ├── windowsrule.conf  # Regras de janelas
    ├── layout.conf             # Workspaces e layouts
    ├── binds.conf              # Teclas de atalho
    ├── misc.conf               # Configurações diversas
    └── autostart.conf        # Aplicações de inicialização
